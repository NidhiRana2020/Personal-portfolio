## 🚀 Quick Start

### Prerequisites
- Node.js 20+
- npm

### Installation
```bash
git clone https://github.com/NidhiRana2020/portfolio-website.git
cd portfolio-website
npm install
```

### Development
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```
